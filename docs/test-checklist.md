# テスト / 回帰確認チェックリスト

デプロイ前や大きな機能追加後に一通り実施する想定のチェック項目です。  
時間がない場合は「必須」のみを確認し、余裕があれば「推奨」も行ってください。

## 前提
- `npm run dev` または本番と同等の環境でアプリが起動していること
- 必要な環境変数が設定され、API 通信が正常に行えること

## 必須チェック
1. **トップページ表示**
   - ページがエラーなく表示されること
   - おすすめ / 新作 / メーカー別セクションが正しく読み込まれること
2. **検索フロー**
   - 検索バーにキーワードを入力し、候補が表示されること
   - 矢印キーで候補を移動し、Enter またはクリックで詳細ページに遷移できること
   - ホバー時に浮遊コメントが流れない（コメントが無い場合は何も表示されない）ことを確認
3. **詳細ページ**
   - 概要 / 画像が表示され、エラーや 404 にならないこと
   - 戻る操作後に同じゲームに遷移すると、表示がスムーズに行われること（プリフェッチ確認）
4. **カテゴリページ**
   - カテゴリフィルタの選択 / 解除でリストが更新されること
   - ホバー時に詳細ページへ遷移しても問題がないこと
5. **エラーハンドリング**
   - 通信エラーを想定し、ネットワークをオフにした状態で分かりやすいメッセージが表示されること（可能であれば）

## 推奨チェック
1. **無限スクロール / 追加読み込み**
   - 新作ゲームやメーカー別リストで「もっと見る」が期待通りに動作すること
2. **コメント表示**
   - コメントが存在するゲームで浮遊コメントが表示されること
3. **スマートフォンサイズ**
   - モバイルビューでヘッダー・メニューが崩れないこと
4. **アクセシビリティ**
   - キーボード操作だけで主要な操作が可能か軽く確認する
5. **パフォーマンス**
   - Chrome DevTools の Performance / Lighthouse で大きな劣化がないかざっくり確認する

## メモ
- 追加で気づいた不具合や気になる点は `docs/` 以下にメモを残したり、イシューを作成することを推奨します。
- テスト観点が増えた場合はこのファイルに追記してください。
